About Me

Hi! I'm Ahmed, an educator and data enthusiast passionate about teaching practical programming and analytics skills. Through hands-on learning and real-world examples, I aim to make technical knowledge accessible and empowering.

About LearnProgrammingWithZee

LearnProgrammingWithZee is an independent digital learning initiative that provides free, project-based tutorials in Visual Basic, data automation, and reporting. We're not a registered organisation — just a growing platform built to share skills that matter in the real world.

What You'll Learn

Through this series, you’ll build a complete VB.NET 9 console application that interacts with Excel. Along the way, you’ll explore:

  • Core programming concepts in Visual Basic
  • Data input, validation, and file interaction
  • Using SQL for querying and analyzing data
  • Automating Excel reports and dashboards using VBA

By the end, you’ll have a fully functioning app that bridges code and spreadsheets — ideal for analytics and automation tasks.
