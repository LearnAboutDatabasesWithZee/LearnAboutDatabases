📄 Doc Data Extractor Chatbot

by Learn Data Analysis With Zee

Welcome to a hands-on project from Learn Data Analysis With Zee — where code meets clarity. This chatbot helps you extract structured data from Word documents using regex, and is perfect for managing and analyzing text-based expense records.

🎯 Project Purpose

This project is designed to streamline data extraction from an organized folder structure used for expense tracking:

  • /Expenses Tracker/ – Root folder
  • ├── Monthly subfolders (e.g. Jan2025, Feb2025)
  • │ └── Daily sub-subfolders (e.g. 01Jan2025, 02Jan2025)
  • │ └── expenses.docx – Daily record of spending
By uploading a .docx file from any day’s folder, the chatbot reads its text, identifies key patterns, and formats them into query-ready tables—making analysis and aggregation effortless.

🛠️ What You'll Build

  • 🗣️ Chatbot-style document queries
  • 📄 Upload and view Word file content
  • 🔍 Extract structured data using modular regex functions
  • 📊 Display data in formatted tables for future reporting

📁 Tech Directory Structure

Here’s how your codebase will be structured:

  • /doc_extractor_bot/
  • ├── app.py – Flask routes
  • ├── /templates/index.html – Frontend layout
  • ├── /static/ – CSS & JS for UI and interactivity
  • ├── /extractors/ – Regex modules by data type
  • └── /utils/file_to_text.py – .docx to text converter
