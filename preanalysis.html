ðŸ”— Step 3: Add a Connection String

In your WinForms project, open Settings.settings and add a new entry:

  • Name: DbConnectionString
  • Type: (Connection String)
  • Scope: Application
  • Value (example):

Data Source=(localdb)\MSSQLLocalDB;Initial Catalog=ExpensesDb;Integrated Security=True;

Adjust the server name or instance if using SQL Server Express instead of LocalDB.
